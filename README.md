# Shopping
KFood_Shopping App

Shopping앱 프로젝트는 한국음식을 배달 App 처럼 먹고싶은 음식을 고른후에 Apple Pay로 결제할수 있도록 구현한 iOS App 프로젝트 입니다

### 이런 기능이 있습니다
- 배달 App처럼 먹고싶은 한국음식을 장바구니에 추가후 최종금액을 Apple Pay로 결제하여 이용할수 있는 기능이 있습니다.
- (현재 Apple Pay는 한국에서 런칭되지 않아 사용할수 없으며, 현재 Test용으로 개발중입니다)

### 사용 인터페이스 및 기술
- SwiftUI
- Apple Pay(현재 개발 구현중)

### Apple Pay 라이브러리(API)
- PassKit

### 사용 언어
- Swift

### 사용 프로그램
- Xcode 13.3
